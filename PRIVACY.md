# Privacy Policy

This application is operated for personal, single-user use only by the
repository owner (the "Operator"). It is not offered as a service to
any third party.

## Scope

The application accesses bank account data exclusively via the Enable
Banking API in restricted mode. It can only retrieve data from bank
accounts that the Operator has explicitly linked and whitelisted in
the Enable Banking portal. No other users exist or can exist.

## Data handled

- Account metadata (account holder name, account number, currency)
- Account balances
- Transaction history for the linked accounts

## Data storage

All retrieved data is stored on self-hosted infrastructure controlled
solely by the Operator. Data is not transmitted to any third party,
not shared, not sold, and not used for analytics, advertising, or
profiling.

## Retention and deletion

Data is retained only as long as the Operator finds it useful for
personal budgeting and financial planning. The Operator may delete
stored data at any time by removing the underlying databases and
revoking the application's access in the Enable Banking portal.

## Contact

Questions about this policy can be raised as an issue on this
repository.
